import draw
Must be at the top. Provides access to all drawing and event tools.
Creating a Canvas
cv = draw.Canvas(width, height)
Creates a new canvas element. You can create as many as you want!
Ex: cv = draw.Canvas(200, 200)
cv.rect() / cv.circle() / ...
All drawing methods below can be called on a specific canvas instance.
Drawing Commands
draw.rect(x, y, w, h, color)
Draws a rectangle. Color can be a string, a hex value or an rgba() object.
Returns a Shape object.
Ex: draw.rect(50, 50, 25, 25, "red")
draw.circle(x, y, radius, color)
Draws a circle at center (x, y).
Returns a Shape object.
Ex: draw.circle(50, 50, 25, "#32a852")>
draw.line(x1, y1, x2, y2, color, width)
Draws a line with a specific thickness.
Returns nothing.
Ex: draw.line(50, 50, 25, 25, "red", 10)
draw.text(val, x, y, color)
Draws text. val can be a string, number, or variable.
Returns nothing.
Ex: draw.text("Some Text", 50, 50, "red")
draw.image(path, x, y, width, height)
Loads image from URL or Filesystem. Automatically handles async loading.
width or height are not necessary
Returns a Shape object.
Ex: cv = draw.image("map.png", 50, 50)
draw.gradient(c1, c2)
Creates a vertical fade.
Can be used in other methods as the color
Returns a color object.
Ex: draw.rect(0,0,100,100, draw.gradient("blue", "black"))
draw.set_alpha(0.0 - 1.0)
Sets transparency for all following shapes on that canvas.
Returns nothing.
Ex: draw.set_alpha(0.5)
draw.clear()
Wipes the canvas drawings and shapes, but keeps keys active.
Returns nothing.
Ex: draw.clear()
Events & Interaction
draw.on_key("key", func)
Runs func when a key is pressed.
Keys: " ", "ArrowUp", "a", "Enter", etc.
Returns nothing.
Ex: def d_pressed:
    print("D was pressed!")
draw.on_key("D", d_pressed)
draw.is_key_pressed("key")
Returns True/False.. Use inside while loops for smooth movement.
Ex: if draw.is_key_pressed("ArrowLeft"): x -= 5
draw.on_click(func)
Global click listener. Function must accept x, y:
Returns nothing.
Ex: def global_click(x, y):
    print(f"Canvas 1 clicked at: {x}, {y}")
    cv.circle(x, y, 10, "rgba(255, 255, 255, 0.5)")
cv.on_click(global_click)
shape.on_click(func)
Attaches a click event only to that specific shape or image.
Returns nothing.
Ex: btn = draw.rect(...); btn.on_click(my_func)
draw.reset()
Destroys all extra canvases and wipes all key/click listeners. Use for a "Hard Restart".
Returns nothing.
Ex: draw.reset()
